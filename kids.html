Kids

Classes will be conducted online using google meet application

Our Flagship Coding Bootcamp for Kids and Teens involves the basics of Commandline, an introduction to web development and creating and using programming fundamentals in Python and Javascript.Lessons will take place in the weekends between 7AM- 12PM in 2 sessions 7AM - 9AM and 10AM - 12PM

Course Topics for the Flagship

  1. HTML and CSS (Duration 2 Months - 20 hrs)

    • Text editors
    • Command line
    • HTML page layout elements
    • HTML tags
    • CSS basics
    • Styling with classes
    • Debugging CSS and HTML
    • Building Games with CSS
    • CSS Animations and Games

  2. Javascript and Python for Kids (Duration 2 Months- 20 hrs)

    • Introduction to JavaScript
    • JavaScript console
    • Fundamental JavaScript
    • Writing functions
    • Business vs User interface logic
    • Control flows
    • Logical operators
    • Arrays
    • Looping

  3. JavaScript for Web and Front-End Development (Duration 2 Months - 20 hrs)

    • Bootstrap
    • Navigating bootstrap documentation
    • Mockup design to website with bootstrap
    • DOM Manipulation techniques
    • Forms
    • Debugging javascript
    • This keyword
